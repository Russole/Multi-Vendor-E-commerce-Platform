# Multi-Vendor-Ecommerce-Platform

## Deploy To AWS

E-commerce Platform : https://emecommerce.com/  
Merchant platform : https://emecommerce.com:3001/login

## How To Run

### FrontEnd

```
cd frontend
npm install
npm start
```

### Dashboard

```
cd Dashboard
npm installs
npm start
```

### BackEnd

```
cd Backend
npm install
npm run server
```
